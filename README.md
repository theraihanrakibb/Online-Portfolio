# Portfolio Website

Single-page portfolio for **Md Rakibul Islam Raihan** with a clean, modern UI, dark/light mode, animated hero background, scroll reveal transitions, typed role effect, and animated counters.

## Structure

- `index.html` - semantic page layout and content
- `css/style.css` - core design, responsive layout, glassmorphism, cards
- `css/animations.css` - reveal and micro animation styles
- `css/dark-mode.css` - dark theme variables
- `js/main.js` - navigation, active links, mobile menu, theme persistence
- `js/scroll-animations.js` - intersection reveal + hero canvas network animation
- `js/counter.js` - stat counter animation on visibility
- `js/typed-effect.js` - typed rotating roles in hero

## Run

Open `index.html` directly in your browser, or serve with any static server.

## Deployment

Deploy to GitHub Pages, Netlify, or Vercel as a static site.
